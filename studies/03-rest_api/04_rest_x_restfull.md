## Status codes

- 100 = infomarmations
- 200 = Success
- 201 = Created
- 204 = No content

- 300 = Redirection

- 400 = Bad request
- 401 = Unauthorized
- 403 = Forbidden
- 404 = Not found

- 500 = Internal server error
